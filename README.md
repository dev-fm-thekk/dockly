# Dockly

Dockly is a lightweight TUI(Terminal User Interface) for visualizing container details without using docker cli or docker desktop
## Authors

- [Abhiram AR](https://www.github.com/dev-fm-thekk)
## License

[MIT](https://choosealicense.com/licenses/mit/)


## Run Locally

Clone the project

```bash
  git clone https://github.com/dev-fm-thekk/dockly
```

Go to the project directory

```bash
  cd dockly
```

Install dependencies

```bash
  go mod download
```

Start

```bash
  go run cmd/dockly/main.go
```

build

```bash
    make build 
```

build-all

```bash
    make build-all
```
